# Security Policy

## Wspierane wersje

Poprawki bezpieczeństwa są wprowadzane tylko do najnowszego wydania.

## Zgłaszanie podatności

**Nie otwieraj publicznego issue na GitHubie w sprawie podatności bezpieczeństwa.**

Zgłaszaj podatności prywatnie przez [GitHub Security Advisories](https://github.com/wielorzeczownik/vscode-polishc/security/advisories/new).

Dołącz jak najwięcej szczegółów:

- Opis podatności
- Kroki do odtworzenia
- Potencjalny wpływ
- Sugerowana poprawka (jeśli masz)

Odpowiedź nastąpi w ciągu **7 dni**. Jeśli problem zostanie potwierdzony, poprawka zostanie wydana jak najszybciej, a Ty zostaniesz wymieniony w release notes (chyba że wolisz pozostać anonimowy).
